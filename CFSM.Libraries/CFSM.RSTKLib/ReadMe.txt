This project library contains CFSM customized excerpts
taken from the RocksmithToolkitLib project library

Do not modify the RocksmithToolkitLib standard project library 
as this may change when the toolkit is updated

Make all customizations here in the CFSM.RSTKLib project files
Do not to use any MessageBoxes in the library (bad practice)
Use the LibLogger to pass messages to the caller
