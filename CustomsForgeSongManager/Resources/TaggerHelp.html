Tagging Songs

Tagging songs allows you to add custom image layers to a CDLC albums artwork. This will only work for CDLC and not official DLC. Tagging requires a "Tagging Theme", which can be created a couple of different ways. The first (Folder Theme) way is to create some png images and put them in a folder, all images need a specific name and it needs a preview png in the root folder. Both theme types have to be placed in "Documents\CFSM\Tagger\templates"

Folder Themes:

  • %THEMENAME%prev.png (goes in the root folder)
  • %THEMENAME%\Background.png
  • %THEMENAME%\Bass.png
  • %THEMENAME%\Custom.png
  • %THEMENAME%\Lead.png
  • %THEMENAME%\Lead Bonus.png
  • %THEMENAME%\Rhythm.png
  • %THEMENAME%\Rhythm Bonus.png
  • %THEMENAME%\Vocal.png
  • %THEMENAME%\DD.png
  • %THEMENAME%\info.txt

 

Tagger Theme XML file format:

This is an xml document that provides drawing instructions. The root name of the xml is "CFSMTaggerTheme". Child elements are DD, NDD, and Custom. The DD element only draws when dynamic difficulty is detected, as NDD draws when there is no dynamic difficulty. The Custom element is always told to draw, but you can add your own conditions to control when and what to draw.

Each of these children have only one attribute, "pos". This is an x,y value that tells where to start the drawing (in pixels). The image that is drawing on is always the largest CDLC artwork dimensions (256x256). The next element will be "Drawing" this too only has one element : "size". The "size" element tells the program to clip the drawing at that size. Here's a example of what has been covered.

<CFSMTaggerTheme xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema">
<DD pos="2,25">
<Drawing size="-1,-1"/>
</DD>
</CFSMTaggerTheme>
